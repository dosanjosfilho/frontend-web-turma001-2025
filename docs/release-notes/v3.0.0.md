# Release v3.0.0 — Interatividade, Acessibilidade e Deploy

## Destaques
- Conversão completa do projeto para uma SPA em JavaScript puro (roteamento hash + templates reutilizáveis).
- Validação avançada de formulários com feedback visual e persistência automática via `localStorage`.
- Máscaras reativas para CPF, telefone e CEP com reaproveitamento ao navegar entre rotas.
- Menu responsivo com dropdown acessível, controle de modal/toast e delegação de eventos centralizada.
- Modo escuro e versão de alto contraste com persistência preferencial e suporte a `prefers-color-scheme`.
- Pipeline de build em Node.js que gera artefatos minificados em `dist/` para publicação (HTML, CSS e JS).
